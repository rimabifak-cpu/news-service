Редактирование настроек
Настройки хранятся в файле .env на сервере.
Для изменения выполните команду на сервере: 
nano /opt/news_service/.env
Команды для изменения настроек:
  1. Подключитесь к серверу по SSH
  2. Выполните: nano /opt/news_service/.env
  3. Измените нужные параметры
  4. Сохраните: Ctrl+O → Enter
  5. Выйдите: Ctrl+X
  6. Перезапустите сервис: docker-compose restart
